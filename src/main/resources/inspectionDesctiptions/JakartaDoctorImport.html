Detecta imports javax.* elegíveis para migração para jakarta.* (Jakarta EE 9+).

Verificações de segurança:

  • Verifica se o pacote javax.* é um que foi migrado para jakarta.* no Jakarta EE 9+
  • Verifica se a biblioteca jakarta.* correspondente está disponível no classpath do módulo
  • Pacotes javax que permanecem no Jakarta EE 9+ (javax.sql, javax.xml, etc.) não são migrados

O quick-fix só será aplicado se a biblioteca Jakarta correspondente estiver no classpath.
