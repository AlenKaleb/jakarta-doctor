Detects imports javax.* eligible for migration to jakarta.* (Jakarta EE 9+).

Security checks:

  • Checks if the javax.* package is one that has been migrated to jakarta.* in Jakarta EE 9+
  • Checks if the corresponding jakarta.* library is available in the module's classpath
  • javax packages that remain in Jakarta EE 9+ (javax.sql, javax.xml, etc.) are not migrated

The quick-fix will only be applied if the corresponding Jakarta library is in the classpath.
